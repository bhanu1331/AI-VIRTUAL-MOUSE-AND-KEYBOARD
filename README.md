🖱️ AI Virtual Mouse & Keyboard

An AI-based system that allows users to control their computer using hand gestures captured via a webcam. This project replaces traditional input devices like a mouse and keyboard using computer vision and real-time gesture recognition.


---

🚀 Features

🖐️ Real-time hand tracking using webcam

🖱️ Control mouse cursor using hand movements

👆 Perform left and right click using gestures

⌨️ Virtual keyboard for touchless typing

⚡ Smooth and responsive interaction

📄 Output demonstration included (OUTPUT.pdf)

⚙️ Installation

1. Clone the repository:



git clone https://github.com/bhanu1331/AI-VIRTUAL-MOUSE-AND-KEYBOARD.git
cd AI-VIRTUAL-MOUSE-AND-KEYBOARD

2. Install dependencies:



pip install opencv-python mediapipe pyautogui numpy


---

▶️ Usage

Run mouse control:

python Mouse.py

Run keyboard control:

python Keyboard.py


---

✋ Gesture Controls

Gesture	Action

Index finger movement	Move cursor
Index + thumb touch	Left click
Two fingers up	Right click
Hand gestures	Keyboard typing



---

⚙️ How It Works

Webcam captures real-time video

MediaPipe detects hand landmarks

Finger positions are tracked

Gestures are mapped to actions

PyAutoGUI executes mouse and keyboard controls
